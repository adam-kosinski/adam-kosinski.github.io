Interactive Graph Editor

This is a program I created when studying mathematical graphs in high school. It allows you to input a graph and move around the vertices, a useful tool for identifying if two graphs are the same. It also automatically generates the adjacency matrix of the graph, which can be copied and used by other programs to analyze the graph. This is a much quicker method of inputting graphs into a computer than creating the adjacency matrix by hand.
